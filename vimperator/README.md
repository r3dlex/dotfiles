# Vimperator Configuration (Archived)

Legacy Firefox Vimperator configuration. Vimperator is no longer maintained and does not work with modern Firefox.

For vim-like browsing in modern browsers, consider:
- [Vimium](https://chromewebstore.google.com/detail/vimium/dbepggeogbaibhgnhhndojpepiihcmeb) (Chrome/Edge)
- [Tridactyl](https://addons.mozilla.org/en-US/firefox/addon/tridactyl-vim/) (Firefox)
